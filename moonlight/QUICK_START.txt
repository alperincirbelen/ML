📦 INSTALLATION (5 minutes)
────────────────────────────────────────────────────────────────────────────

1. Open PowerShell/CMD in moonlight folder

2. Install Python dependencies:
   pip install -r requirements.txt

3. Create your config:
   copy configs\app.example.yaml configs\app.yaml

4. Initialize database:
   python -c "import asyncio; from moonlight.core.storage import Storage; asyncio.run(Storage('data/trades.db').init()); print('✓ DB ready')"


🎮 FIRST RUN (2 minutes)
────────────────────────────────────────────────────────────────────────────

Option 1 - Quick Test:
   python quick_test.py

Option 2 - Paper Trading (10 min):
   python run_paper.py --duration 10

Option 3 - Full Engine:
   python -m moonlight.core.main --config configs/app.yaml


🖥️ USER INTERFACE (Optional)
────────────────────────────────────────────────────────────────────────────

1. Open new terminal in ui_app folder

2. Install Flutter dependencies:
   flutter pub get

3. Run desktop app:
   flutter run -d windows


📡 API ACCESS
────────────────────────────────────────────────────────────────────────────

Start API server:
   python -m moonlight.core.api.server

Visit: http://127.0.0.1:8750/docs

🔧 CONFIGURATION
────────────────────────────────────────────────────────────────────────────

Edit: configs/app.yaml

Key settings:
  mode: paper                    # paper | live
  connector: mock                # mock | olymp
  ensemble_threshold: 0.70       # Decision threshold
  
  accounts:                      # Your accounts (1-4)
    - id: acc1
      username: your@email.com
      
  products:                      # Trading pairs
    - product: EURUSD
      strategies: [5, 14, 15]    # Strategy IDs
      
  limits:                        # Safety
    max_daily_loss: 5.0
    max_consecutive_losses: 5
